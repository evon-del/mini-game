# How To Play
1.Upon starting, the game will prompt you to enter a guess between 1 and 100.

2.Type your number and press Enter.

3.Pay attention to the hints!

4.If it says Too low, try a higher number.

5.If it says Too high, try a lower number.

6.You win if you guess the exact number within 7 attempts.

7.After the game ends (whether you win or lose), your current score will be displayed.

8.You will be asked if you want to play again. Type Y for Yes or N for No.

# Configuration
You can easily adjust the game difficulty by modifying the constants at the top of the script:

- MIN_NUMBER: The lowest possible secret number (default: 1)

- MAX_NUMBER: The highest possible secret number (default: 100)

- MAX_TRIES: The number of guesses allowed per round (default: 7)
